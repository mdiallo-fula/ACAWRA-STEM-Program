Overview

The STEM Program will roll out in stages, beginning with Math tutoring sessions on December 2. Parents who register their children will receive updates as additional tutoring sessions in Physics, Web Development, and Arduino (robotics) become available. In addition to the tutoring sessions, the program will also offer a recreational activity: basketball games. These games will be open to all high school students enrolled in the program. An indoor basketball court will be provided for 2-3 times a week. All program activities will take place Monday through Friday, from 5 PM to 8 PM, at 675 Queen St S, Kitchener, ON N2M 1A1, except for the basketball games. Further details will be sent to registrants during the weekend leading up to December 2.

Courses

Math & Physics

  • Get help mastering concepts taught at school
  • Develop effective study strategies and exam preparation techniques
  • Gain advanced problem solving skills through contest preparation (math only)

Web Development

  • Intoduction to programming with Python (grades 7-9) / JavaScript (grades 10-12)
  • Learn how to create responsive web pages with HTML and CSS
  • Add interactive functionalities to websites with JavaScript (grades 10-12)

Arduino (robotics)

  • Learn about the electronic components in an ELEGOO UNO Super Starter Kit
  • Use the electronic componenents to build simple circuits
  • Learn how to use the Arduino IDE to program microcontrolers
